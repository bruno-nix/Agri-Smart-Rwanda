# **App Name**: Agri-Smart Rwanda

## Core Features:

- AI Crop Scanner: Utilize AI to assess crop quality from a photo and suggest a quality grade (A, B, C).
- Dynamic Pricing Engine: Suggest fair market rates (RWF) based on crop type, quality grade, and location; the LLM may use tool(s) for calculation of fair market price.
- Crop Listing Form: Form allowing farmers to upload a photo, select crop type, input quality grade, and location to create a listing.
- Marketplace Feed: Display listed crops with photos, quality grade, price per KG, and contact information in a feed.
- Bilingual Interface: Toggle to switch between English and Kinyarwanda for all text elements.

## Style Guidelines:

- Background color: Light green (#E8F5E9), subtly evokes trust and growth without overwhelming the interface.
- Primary color: Forest green (#388E3C), reflects agricultural theme and highlights key interactive elements.
- Accent color: Yellow (#FDD835), draws attention to call-to-action buttons.
- Body and headline font: 'PT Sans', a humanist sans-serif for a modern, readable style across both English and Kinyarwanda.
- Use clear, recognizable icons to represent crop types and quality grades, ensuring usability for users with varying literacy levels.
- Mobile-first design with large, easily tappable buttons for farmers using smartphones in the field.
- Subtle loading animations during AI analysis to provide user feedback without being distracting.
